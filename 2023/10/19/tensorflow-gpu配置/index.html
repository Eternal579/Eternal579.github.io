预备知识

安装python，本质上就是在电脑上安装了一个文件夹，里面有一个最关键的程序python.exe，这个程序用来interpret你写的python代码。平时pip install abracadabra的时候，会把相应的包安装在python*.*.*/Lib/site-packages中

当本地安装了多个版本的python时，只需要修改环境变量便可切换不同的版本
如下图，更改系统变量的Path变量即可

如此，在cmd中输入python，便会发现版本已被切换

与此同时记得修改VsCode中的python interpreter,通过ctrl+shift+p输入Python:Select interpreter来修改

安装Tensorflow

先安装CUDA

  • 使用nvidia-smi查看你的显卡所能支持的最高版本CUDA 注意！这里只是显示支持的版本，不代表已经安装了CUDA，只是安装了CUDA的驱动而已
  • 前往Tensorflow官网查看所需版本，这里选的如下版本（记住版本宁旧勿新！）
  • 然后去CUDA下载就好了，注意下像前面装过驱动的就可以不勾选Driver components了

然后安装cuDNN

  • 然后去cuDNN下载相应版本的cuDNN，然后把cuDNN文件下的bin,inclue,lib复制进CUDA文件夹下的bin,inclue,lib

最后安装tensorflow

  • 然后使用pip install tensorflow==2.4.0安装所需版本的tensorflow

需要注意的两个点

  • tensorflow2.4版本，默认不注册XLA，所以需要使用时在代码开头加上 不然你就会遇到
  • 修改CUDA文件夹下的bin文件夹下的这个dll，原本结尾为11，需要修改为10
